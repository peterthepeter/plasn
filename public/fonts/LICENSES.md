Bundled web fonts in this directory:

- Public Sans — SIL Open Font License 1.1 — https://github.com/uswds/public-sans
- Source Code Pro — SIL Open Font License 1.1 — https://github.com/adobe-fonts/source-code-pro
- JetBrains Mono — SIL Open Font License 1.1 — https://github.com/JetBrains/JetBrainsMono

These files are bundled locally so both container images can render the UI and browser print views without external font requests.
